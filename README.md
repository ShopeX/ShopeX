# ShopeX：TECHNOLOGY DRIVES THE FUTURE OF BUSINESS
ShopeX成立于2002年，距今已有23年的数智化服务经验，目前已是众多全球头部知名品牌和集团企业的“数智科技伙伴”。商派以Open SaaS模式提供一站式、一体化的数智化产品与服务，主要包括：数智化轻咨询服务,2C交易系统,2B业务平台,全渠道OMS系统,微服务DigiOS数智化操作系统，AI解决方案与运维保障服务等。  
ShopeX的产品和项目管理均获得了国际权威机构的官方认证：如ISO9001、 ISO27018、 ISO27001、DJCP(等保三级）等，可以有效确保系统的安全与稳定。  
商派一直是开源领域的积极推动者和重要贡献者，早在2008年就发布了ECshop开源项目。如今，商派再次将公司两大核心产品ECShopX（分布式交易系统）与ONEX OMS（全渠道订单管理系统）进行全面开源，通过持续的技术共享和生态建设，推动中国零售电商行业的技术进步和创新发展，赋能更多品牌企业。
